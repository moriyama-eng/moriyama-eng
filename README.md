### Hi there 👋 I'm Moriyama

While working as a mechanical design engineer in the manufacturing industry, I build Monte Carlo simulators to quantify financial risks and pursue a **"peaceful FIRE."**

I apply the concepts of **redundancy** and **safety factor**—cultivated in mechanical design—to asset management. This repo showcases my work in probabilistic financial modeling.

## 🛠️ Tech Stack & Tools
- **Simulation:** Monte Carlo simulations utilizing the **Geometric Brownian Motion (GBM)** model for future asset projection.
- **Frontend:** HTML, JavaScript (Chart.js), Tailwind CSS

## 🚀 Projects
- **[FIRE Monte Carlo Simulator](https://moriyama-eng.github.io/fire-simulator/)**
  - A web application that probabilistically verifies the effectiveness of cash buffers by simulating the **"sequence of returns risk"** in post‑FIRE asset progression using random variables.
